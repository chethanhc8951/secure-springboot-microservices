# Secure Spring Boot Microservices Architecture

A production-style microservices project built using **Spring Boot 3, Java 21, Spring Security, JWT, Eureka Service Discovery, API Gateway, Docker, and MySQL**.

This project demonstrates a **secure, scalable microservices architecture** where multiple services communicate through service discovery and are accessed through a centralized API Gateway.



## 🚀 Features

- Microservices architecture using Spring Boot
- Service discovery using Eureka Server
- Centralized routing with API Gateway
- JWT based authentication and authorization
- Role-Based Access Control (RBAC)
- Admin and User access levels
- Product management with CRUD operations
- Containerized deployment using Docker
- MySQL database integration
